• HTML is a markup language for the web that defines the structure of web pages. Learn what HTML tags, elements, attributes, and semantics are and how they work together to create web pages.

  • CSS, or Cascading Style Sheets, offers a flexible way to style web content, with styles originating from browser defaults, user preferences, or web designers.

  • JavaScript (JS) is a lightweight interpreted (or just-in-time compiled) programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat.

  • React is a JavaScript library created by Facebook for building user interfaces (UIs). It allows developers to create reusable UI components, which can be combined to build complex applications. React is widely used in web development, mobile app development (via React Native), and even desktop app development (via Electron).

  • Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser It is built on the Google Chrome V8 JavaScript engine, which makes it highly performant Node.js is widely used for building scalable network applications and is popular among companies like Netflix and Uber.
